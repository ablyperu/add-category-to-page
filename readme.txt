== Description ==
Plugin that allow to add category into wordpress page, you can filter and more.

= Features =
* add post categories to page
* add tags to page
* filter categories

== Changelog ==
= Version 1.0 =
* Initial Plugin release

== Installation ==
It is better to install from the wordpress repository, in case you can not install, you can manually download and install it in the “wp-content/plugins”.
